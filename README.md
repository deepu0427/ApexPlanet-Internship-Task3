# Web Development Practice Projects

## 📌 Overview
This repository contains a set of practice projects I built to strengthen my **frontend development** and **JavaScript** skills.  
The projects focus on responsive design, interactivity, and working with external APIs.

---

## 🚀 Projects

### 1. Responsive Design Using Media Queries
**Objective:** Ensure web pages look great on mobile, tablet, and desktop.  
**Actions:**
- Applied CSS media queries to adjust layouts, font sizes, and elements.  
- Tested across different screen sizes for a consistent user experience.  

---

### 2. Interactive Quiz & Image Carousel (JavaScript)
**Objective:** Build more complex JavaScript projects.  
**Actions:**
- Created an interactive quiz with multiple questions and dynamic feedback.  
- Developed an image carousel that rotates images on user interaction.  

---

### 3. Fetch Data from an API Using JavaScript
**Objective:** Learn how to work with external APIs to fetch real-time data.  
**Actions:**
- Connected to public APIs (e.g., weather or joke API).  
- Displayed fetched data dynamically on the webpage.  

---

## 🛠️ Tech Stack
- **HTML5**  
- **CSS3 (Media Queries)**  
- **JavaScript (DOM Manipulation, API Fetch)**
